SAP BTP Travel Management System
About This Project
A modern, cloud-ready Travel Management System built on the SAP Business Technology Platform (BTP). This project was developed as a practical case study for my Master of Science thesis. It shows how to build a clean, powerful enterprise application using Modern ABAP and SAP Fiori.

Built Using:
1. Backend: SAP Business Technology Platform (BTP)
2. Programming Model: ABAP RESTful Application Programming Model (RAP)
3. Development Tool: Eclipse (ABAP Development Tools)

Key Features:
1. Create Travel Requests: Users can easily plan business trips by entering customer details, booking flights, and adding extras (like luggage or meals).
2. Auto-Save (Draft Mode): The app automatically saves your work in the background. If you close your browser, you will not lose your data. You can come back and finish the trip later.
3. Live Price Updates: The system instantly calculates and updates the total price of the trip whenever you add or remove a flight or extra item.
4. Smart Error Checking: The app automatically stops users from making mistakes. For example, it will show an error if you try to book a flight for a date in the past.
5. Manager Approvals: Managers have special buttons to review trips and click 'Accept' or 'Reject'. Once a manager makes a decision, the trip is locked and cannot be changed.
6. File Uploads: Employees can easily upload and attach files directly to their trip, such as PDF flight tickets or hotel receipts.
7. Copy Past Trips: To save time, users can click a "Copy" button to instantly duplicate an old trip (including all its flights and extras) to create a new one.
8. Apply Discounts: Managers have a custom action button to quickly apply a percentage discount to the agency booking fee.
9. Automatic Database Cleanup: If a user deletes a travel request, the system is smart enough to automatically delete all the attached flights and files so the database stays perfectly clean.
